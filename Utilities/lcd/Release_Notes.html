Purpose

This driver provides a set of basic 2D display functions common for all LCDs. It can be used on top of BSP LCD drivers offering the following basic APIs:

  • BSP_LCD_DrawBitmap
  • BSP_LCD_FillRGBRect
  • BSP_LCD_DrawHLine
  • BSP_LCD_DrawVLine
  • BSP_LCD_FillRect
  • BSP_LCD_ReadPixel
  • BSP_LCD_WritePixel
  • BSP_LCD_GetXSize
  • BSP_LCD_GetYSize
  • BSP_LCD_SetActiveLayer
  • BSP_LCD_GetPixelFormat
